PROJECT 01 | Shopify Operations

Shopify Operations Dashboard and Order Issue Tracker

A store operations system built to review order status, fulfillment blockers, supplier remarks, support notes, COGS, and Shopify sales performance in one place.

Project

Shopify operations visibility setup for a store handling fulfillment, supplier issues, customer remarks, and sales reporting.

The Problem

The store had order issues spread across Shopify views, supplier messages, customer support notes, and manual spreadsheets. Fulfilled, unfulfilled, waiting for customer response, cancelled orders, supplier remarks, and COGS needed one place for review.

The Solution

Created an operations tracker with order status labels, issue categories, supplier remarks, store manager notes, customer support manager notes, fulfillment date, review status, and summary counts.

Execution

Tracked 178 fulfilled orders, 10 unfulfilled orders, 5 orders waiting for customer response, 2 cancelled orders, and $8,673.40 total COGS. Added dropdown statuses for shipping address issues, VAT numbers, size issues, order cancellation, and other blockers.

Result

The operations view made it easier to see which orders needed supplier action, which issues needed store manager review, and which orders were already checked. The Shopify analytics view also captured $1,321,820.57 gross sales, $1,200,037.67 total sales, 17,991 fulfilled orders, and 18,251 orders.

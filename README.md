# James Wilher Atenta Portfolio

GitHub-ready static portfolio for Shopify Operations Manager and Social Media Manager roles.

## Files to upload

Upload these to the root of the repository:

```
index.html
styles.css
script.js
README.md
media-v4/
```

## Important image note

This version does not use the old `assets/` folder. It uses `media-v4/` so GitHub Pages cannot load old mixed image files.

Do not rename `media-v4`. The HTML points to exact filenames inside that folder.

If your old repository still has `assets/`, it will not affect this version because the page no longer uses that folder.
